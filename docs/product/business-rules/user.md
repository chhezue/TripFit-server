# User (계정·프로필) 비즈니스 규칙

> 출처: Figma Wireframe v1.

## BR-USER-001

- **조건:** 회원가입/로그인
- **동작:** Google, Kakao, Apple 소셜 OAuth 지원
- **위반 시:** 401

## BR-USER-002

- **조건:** 가입 직후 캘린더 연동 안내
- **동작:** Google/Naver 연동 또는 `나중에 하기` 허용 (연동 필수 아님)
- **위반 시:** —

## BR-USER-003

- **조건:** 회원 탈퇴
- **동작:** 확인 후 탈퇴; **내가 만든 여행방 데이터 전부 삭제** (와이어프레임 카피)
- **위반 시:** 409, [미정] 진행중 여행방 존재 시 차단 여부

## BR-USER-004

- **조건:** 알림 허용 설정
- **동작:** on/off 저장, off 시 푸시 미발송 [미정: 필수 알림 예외]
- **위반 시:** —

## BR-USER-005

- **조건:** 기본정보 저장
- **동작:** 근무·연차 프로필 upsert; 이후 추천 계산에 반영
- **위반 시:** 400
