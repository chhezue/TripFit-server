# 추천 일정 확정 플로우

> 출처: Figma `추천일정` 섹션.

- **목적:** 멤버 응답 기반 최적 여행 기간 확정
- **액터:** 방장 또는 멤버 [미정: 확정 권한]
- **사전 조건:** 여행방 존재, 일부 이상 멤버 응답 (BR-REC-001)
- **단계:**
  1. 여행방 상세 → 추천 일정 탭
  2. 응답률 확인, 필요 시 응답 요청 (BR-SCHEDULE-005)
  3. 추천 전략 선택 (BR-REC-002)
  4. 추천 일정보기 → 후보·근거 확인 (BR-REC-003)
  5. 일정 확정하기 (BR-REC-004)
- **성공 종료:** 상태 `일정 확정`, 확정 기간 저장
- **예외:** 응답 부족 → 422
- **MVP:** In
