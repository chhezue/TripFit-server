# 여행방 생성 플로우

> 출처: Figma `여행방 생성`. MVP 포함.

- **목적:** 새 여행방 개설 및 방장 첫 일정 입력
- **액터:** 방장(생성자)
- **사전 조건:** 로그인
- **단계:**
  1. 이름·희망 기간·희망 일수·참여 인원 입력 (BR-ROOM-001)
  2. 여행지 정했어요/못 정했어요 선택 (BR-ROOM-002)
  3. 만들기 → 여행방 생성 (BR-ROOM-003)
  4. 희망 기간 캘린더에서 가용 일정 입력 (BR-SCHEDULE-001)
- **성공 종료:** 홈 또는 여행방 상세 진입, 상태 `응답대기중`
- **예외:** 필수값 누락 → 400
- **MVP:** In
